# マイルストーン

問題:ソフトウェア開発で作成したガントチャートにマイルストーンを適用してください。
マイルストーンとは、プロジェクト管理において、プロジェクトの重要な節目や到達点を表す目印となる出来事や状態のことを指します。例えば、プロジェクトの開始日や終了日、重要なタスクの完了日、決定事項の承認日などがマイルストーンにあたります。
happens でマイルストーンを定義できます。
