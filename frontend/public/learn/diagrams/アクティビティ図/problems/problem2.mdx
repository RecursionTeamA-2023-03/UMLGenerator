# FizzBuzz

1 から 100 までの数字を順番に出力し、3 で割り切れる場合は "Fizz"、5 で割り切れる場合は "Buzz"、3 と 5 で割り切れる場合は "FizzBuzz" と出力します。このアクティビティ図を作成してください。
