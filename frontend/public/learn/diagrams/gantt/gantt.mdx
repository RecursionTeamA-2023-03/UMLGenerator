# ガントチャート

## ガントチャートとは？

ガントチャート（Gantt chart）は、作業やタスクの進捗状況を時系列で視覚的に表現するために用いられるグラフの一種です。横軸に時間、縦軸にタスクや作業の一覧を表示し、それぞれのタスクがどの期間に実行されるかを棒グラフで表現します。各タスクの開始日と終了日を表す棒グラフが並べられることで、プロジェクト全体の進捗状況を一目で確認することができます。

ガントチャートは、プロジェクトマネージャーやチームリーダーがプロジェクトを管理し、スケジュールを立てる上で広く用いられています。また、プロジェクトメンバーが作業の進捗状況を把握し、スケジュールを共有する上でも重要なツールとなっています。

## 基本的な例

テキスト形式でガントチャートを定義することができます。タスクの名前、開始日、終了日、進捗状況を記述します。
コードの最初に@startgantt、最後に@endgantt と書くことでガントチャートを記述できます。
タスクの定義は[Task]のようにに角カッコを使用します。
期間の定期は last を使用します。
開始は start, 終了は end を使用します。また、両方を日付で指定することもできます。

```
@startgantt
language ja

[プロトタイプを設計] lasts 15 days
[プロトタイプをテスト] lasts 10 days

Project starts 2023-07-01
[プロトタイプを設計] ends 2023-07-15
[プロトタイプをテスト] ends 2023-07-25

@endgantt
```

and で繋ぐことで、宣言を簡潔に記述できます。

```
@startgantt
language ja

Project starts 2023-07-01
[プロトタイプを設計] starts 2023-07-01 and ends 2023-07-15
[プロトタイプをテスト] starts 2023-07-16 and lasts 10 days
@endgantt

```
