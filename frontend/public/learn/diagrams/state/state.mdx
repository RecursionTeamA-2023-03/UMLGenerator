# ステート図

## ステート図とは？

ステート図では、オブジェクトが取りうる状態（ステート）をボックスで表し、状態間の遷移を矢印で表現します。

## 基本的な例

ステート図の始点と終点は、 [*] で示します。
状態は state で示します。ボックスの中には、状態名や入出力アクション、入出力イベント、遷移条件などを記述することができます。
遷移の矢印は、 --> で示します。遷移先には状態名を指定します。

```
@startuml

[*] --> State1
State1 --> [*]
State1 : this is a string
State1 : this is another string

State1 -> State2
State2 --> [*]

@enduml

```
