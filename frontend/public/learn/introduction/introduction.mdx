# 学習ページ概要

このページでは、PlantUML の基本的な構文や使用方法を学ぶことができます。また、PlantUML を使用して実際にダイアグラムを作成するための具体的な例を紹介しています。さらに、PlantUML を活用するためのヒントやトリックについても紹介しているので、PlantUML を使いこなすための手助けになるでしょう。

まずはじめに、PlantUML で UML 図を書くためには、"@startUML"で図の始まり、"endUML"で図の終わりを示します。簡単なシーケンス図を書いてみましょう。シーケンス図は、オブジェクト間のやり取りを表します。例えば、オブジェクト A がオブジェクト B に"Hello"と送信するシーケンス図を書いてみましょう。以下はサンプルコードを見て、実際に書いてみましょう。

<EditorPreview
  placeholder="
  
  '@startuml
  'A -> B: Hello
  '@enduml
  '--------------------
  'ここはコメントです。
  'オブジェクトのやり取りは->を使います
  '--------------------
  '下に実際に書いてみましょう。
  "
/>

このように、PlantUML を使用することで、簡単に UML 図を作成することができます。次のページからより詳しく学びましょう。
